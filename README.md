# **红土行歌 · 微信小程序**

## **一、大纲**(目前版本)

### **①小程序结构**

```
    封面
    |
    |--景点列表
    |  |
    |  |--景点1页面（用导航模块整合子页面）
    |  |  |--简介页面
    |  |  |--详细信息页面
    |  |  \--评论区块
    |  |
    |  ...
    |
    \--景点分布地图
```

### **②具体页面结构**

+ **封面**：项目logo + 小程序名称 + 子页面列表 + 其它信息

+ **景点列表页面**：景点名称、景点位置
+ **景点页面**：与三个子页面结合，作为导航模块（参考bilibili），顶部为景点图片与景点名称。
+ **简介页面**：景区概况 & 打卡按钮（位置限制）
+ **详细信息页面**：
  + 景区级别
  + 地址
  + 地图定位 （跳转至地图界面）
  + 当地天气情况
  + 联系电话
  + 营业时间
  + 交通方式
  + 预计游玩时长
  + 其它事项

### **③数据交互**

+ 打卡信息（需获取位置）
+ 评论信息（需获取头像）
