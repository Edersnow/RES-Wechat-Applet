# **红土行歌 · 微信小程序**

## **一、概述**

### **①小程序结构**

```
    主页面
    |
    |--景点1（导航模块）
    |  |--简介页面
    |  |--详细信息页面
    |  \--评论区块
    |
    ...

```

### **②具体页面结构**

+ **主页面**：logo(?)+列表
+ **景点界面**：与三个子页面结合，作为导航模块（参考bilibili），顶部为景点图片。
+ **简介页面**：给出景点大致的介绍，以及打卡按钮
+ **详细信息页面**：
  + 地址
  + 联系电话
  + 当地天气情况
  + 地图导航（tx内置模块）
  + 建议交通方式
  + 建议食宿
  + （待完善）

### **③数据交互**

+ 打卡信息
+ 评论信息
